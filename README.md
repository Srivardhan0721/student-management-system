A C-based console application used to manage student records through file handling, structures, and a menu-driven interface. The system includes role-based access for Admin, Staff, and Guest.

Features:
Role-based login: Admin, Staff, Guest
Add, display, search, update, and delete student records
Uses file handling for permanent data storage
Simple and modular code structure

Description

This project is a simple Student Record Management System developed in C. It demonstrates how to use file handling, structures, and modular programming to perform basic data management tasks. The system includes secure role-based access and provides essential operations needed to manage academic records.
